# mrbeast-skills

A collection of Claude Code skills distilled from MrBeast's content creation methodology — extracted from his internal production guide, podcasts, and decision records.

## Skills

| Skill | Description |
|---|---|
| `mrbeast-perspective` | Roleplay as MrBeast/Jimmy to get direct, opinionated YouTube advice |
| `viral-content-ideation` | Brainstorm video concepts and evaluate viral potential |
| `viral-youtube-hook` | Craft opening hooks that maximize viewer retention |
| `youtube-thumbnail-title-ctr` | Optimize thumbnails and titles for click-through rate |
| `youtube-video-structure` | Structure videos for retention and rewatch value |
| `youtube-video-formats` | Choose the right video format for a given concept |
| `youtube-production-mindset` | Apply MrBeast's production philosophy to any project |
| `brand-deal-integration` | Integrate sponsorships without killing retention |

## Usage

These skills are designed for use with [Claude Code](https://claude.ai/code).

**Install:**

```bash
npx skills add soasme/mrbeast-skills
```

Invoke any skill via the Claude Code `Skill` tool or by typing `/<skill-name>` in the prompt.
